## üßë‚ÄçüöÄ INSTALLATION PROTOCOL

Fluxion is constructed as a secure, independent asset frame. Initialize the environment utilizing standard terminal deployment mechanisms.

### The Best Way (`pipx`)
`pipx` ensures your CLI applications remain isolated without polluting the host environment, while granting global system-wide execution capabilities:
```bash
pipx install git+https://github.com/scorpiocodex/Fluxion.git
```

### The Standard Way (`pip`)
If you prefer developing extensions natively or are compiling inside a strict virtual environment:
```bash
git clone https://github.com/scorpiocodex/Fluxion.git
cd Fluxion
pip install -e .
```
